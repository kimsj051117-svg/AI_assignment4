# LabLog

**LabLog**는 학생 연구자를 위한 한국어 기반 정적 웹 연구노트 서비스입니다. 연구 프로젝트, 문헌, 실험 조건 템플릿, 상황별 실험 템플릿, 연구노트, 실험 비교, 결과 아카이브를 한 화면에서 관리할 수 있습니다.

이 프로젝트는 **Introduction to AI Programming - Assignment 4**를 위해 제작되었습니다.

---

## 주요 기능

### 1. 연구 프로젝트

- 프로젝트명, 연구 분야, 연구 목표, 시작일, 상태, 키워드, 메모 저장
- 프로젝트 카드 표시
- 프로젝트 추가, 수정, 삭제
- 연구노트와 문헌 기록에서 저장된 프로젝트 선택 가능

### 2. 연구 노트 / 새 실험 기록

- 새 실험 기록 작성
- 저장된 프로젝트 선택
- 상황별 실험 템플릿 선택
- 기본 조건 템플릿 선택
- 결과 요약 및 결과 파일 첨부
- 연구노트 카드 표시
- 연구노트 수정, 삭제, 복제
- 검색, 상태 필터, 날짜 정렬

### 3. 상황별 템플릿 라이브러리

기존의 단순한 Template Guide 대신, 실험 상황에 맞는 템플릿을 선택해 새 실험 기록에 적용할 수 있습니다.

포함된 템플릿:

- 일반 실험 노트 템플릿
- 합성 실험 템플릿
- 측정 / 분석 실험 템플릿
- 소자 제작 템플릿
- 조건 최적화 템플릿
- 문헌 기반 실험 계획 템플릿

각 템플릿은 설명, 추천 상황, 주요 항목 미리보기, `이 템플릿 사용하기` 버튼을 제공합니다.

### 4. 실험 조건 템플릿

- 사용자가 기본 실험 변수를 직접 추가/삭제
- 변수명과 기본값을 저장
- 새 실험 기록에서 기본 조건 템플릿을 선택하면 조건이 자동 입력됨
- 기본값과 달라진 변수는 `수정됨`으로 표시

### 5. 변경 변수 추적

LabLog는 기본 조건과 실제 실험 조건을 비교해 변경된 변수만 강조합니다.

예시:

```text
Reaction Temperature: 80 C -> 90 C
Reaction Time: 2 h -> 3 h
```

### 6. 실험 비교

- 저장된 실험 기록을 표 형태로 비교
- 날짜, 실험 제목, 변경 변수, 최종 조건, 결과 요약 확인
- 이전 실험 복제 기능 제공

### 7. 그룹 보기

연구노트 목록에서 보기 방식을 선택할 수 있습니다.

- 전체 기록
- 프로젝트별 보기
- 실험 제목별 보기
- 주요 변수별 보기

그룹 보기에서도 검색, 상태 필터, 날짜 정렬이 적용됩니다.

### 8. 문헌 아카이브

- 논문 제목, 저자, 연도, 저널/출처, DOI 또는 URL, 관련 프로젝트, 키워드, 요약, 개인 노트 저장
- 문헌 기록 추가, 수정, 삭제
- 제목, 저자, 키워드, 프로젝트 기준 검색
- DOI를 입력하고 `DOI 정보 불러오기`를 누르면 Crossref API를 통해 기본 문헌 정보 자동 입력
- URL이 있으면 클릭 가능한 링크로 표시

### 9. 결과 아카이브

- 실험 결과 요약과 첨부 파일 확인
- PDF, Excel, CSV, 이미지 등 결과 파일 첨부
- 결과 기록 수정, 삭제
- 프로젝트명 또는 주요 변수 검색
- 보기 방식에 따라 프로젝트별/제목별/주요 변수별 그룹 표시

### 10. 대시보드 요약

상단 요약 카드에서 다음 정보를 확인할 수 있습니다.

- 전체 노트 수
- 연구 프로젝트 수
- 실험 기록 / 결과 수
- 문헌 / 템플릿 수

## 기술 스택

- HTML
- CSS
- JavaScript
- Browser localStorage

백엔드 서버나 데이터베이스는 사용하지 않습니다.

## 데이터 저장

모든 데이터는 브라우저 `localStorage`에 저장됩니다.

주요 저장 데이터:

- 연구 프로젝트
- 연구노트
- 문헌 기록
- 기본 실험 조건 템플릿
- 실험 기록
- 결과 기록

브라우저 저장소를 삭제하면 저장된 데이터도 삭제됩니다.

## 파일 구조

```text
lablog/
├── index.html
├── style.css
├── script.js
└── README.md
```

### `index.html`

웹사이트의 기본 구조, 사이드바, 상단 탭, 새 실험 기록 모달을 포함합니다.

### `style.css`

흰색 배경과 파란색 포인트를 중심으로 한 카드 기반 UI 스타일을 정의합니다.

### `script.js`

다음 기능을 처리합니다.

- 프로젝트 추가/수정/삭제
- 문헌 추가/수정/삭제 및 DOI 조회
- 상황별 템플릿 적용
- 기본 조건 템플릿 관리
- 연구노트 추가/수정/삭제/복제
- 변경 변수 추적
- 그룹 보기
- 실험 비교
- 결과 아카이브
- localStorage 저장 및 불러오기

## 로컬 실행 방법

### 방법 1: 브라우저에서 직접 열기

1. 프로젝트 폴더를 엽니다.
2. `index.html`을 더블클릭합니다.
3. 웹사이트가 브라우저에서 실행됩니다.

### 방법 2: VS Code Live Server 사용

1. VS Code에서 프로젝트 폴더를 엽니다.
2. Live Server 확장을 설치합니다.
3. `index.html`을 우클릭합니다.
4. **Open with Live Server**를 선택합니다.
